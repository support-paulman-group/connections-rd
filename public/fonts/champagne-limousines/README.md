# Champagne & Limousines

This folder contains the local font files used for the Connections RD brand typography.

## Source

- Font: Champagne & Limousines
- Author: Lauren Thompson / Nymphont
- Download source: https://www.dafont.com/champagne-limousines.font
- Download date: 2026-05-13
- Included project files:
  - `Champagne-Limousines.ttf`
  - `Champagne-Limousines-Bold.ttf`

The original DaFont archive also includes italic variants and `Font-license-readme.txt`. Only the regular and bold upright fonts are used by this site.

## Current Usage

- `Champagne-Limousines-Bold.ttf` is used for the `CONNECTIONS` wordmark through `--font-logo`.
- `Champagne-Limousines.ttf` is used for display headings through `--font-display`.
- Body text remains Montserrat.

## License Note

DaFont lists this font as free for personal use and notes that commercial use requires donation/contact with the author. Because this is a commercial real estate website, the project owner should confirm commercial/web embedding rights before production launch.

Do not describe the commercial license as resolved unless the project owner has confirmed it.

## Replacing The Font Later

If the client provides licensed official files, replace these files while keeping the same names:

- `Champagne-Limousines.ttf`
- `Champagne-Limousines-Bold.ttf`

Keeping the same names avoids CSS changes. After replacement, update this README with the new source, license status, and replacement date.
